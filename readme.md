# thermal-solver

Simple thermal model descriptions in Python, solvable with `scipy.integrate.solve_ivp`.

## Install and run example

```bash
git clone https://github.com/akukulanski/thermal-solver
git clone git@github.com:akukulanski/thermal-solver.git
cd thermal-solver
uv sync --dev

# Run example
uv run python3 examples/test_example_two_nodes.py --show
```
